  What Doesn’t Work / Limitations

• Program assumes CSV structure is always correct. It does not validate malformed lines. 
• User input validation is minimal (non-integer input may cause unexpected behaviour). 
• No exception handling implemented for file I/O failures. 
• Vector does not implement bounds checking for operator[]. 
• Program does not handle extremely large datasets efficiently (resizing is simple doubling strategy). 
• No logging system for error reporting. 
• Solar radiation filtering threshold (>=100) is hard-coded rather than configurable. 
• No graphical output or advanced statistical analysis. 
• Code style is concise; some single-line if statements omit braces for brevity.
